# Project Plan

## Project Overview

The goal of this project is to develop an AI-powered movie recommendation system capable of suggesting relevant movies using both content-based and collaborative filtering techniques.

## Project Roadmap

### Phase 1 – Project Design and Planning

- Define project scope
- Define objectives
- Select technologies
- Design repository structure
- Create documentation

### Phase 2 – Data Collection and Preprocessing
Status: In progress

Tasks:
- [x] Download MovieLens Latest Small dataset
- [x] Store raw data in `data/raw/`
- [x] Explore movies and ratings datasets
- [x] Check missing values and duplicates
- [x] Extract movie year from title
- [x] Clean movie titles
- [x] Convert timestamps to readable dates
- [x] Save cleaned datasets in `data/processed/`
